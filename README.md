# MetaCubeX Rule Mirror

A Vue 3 + Vite static site that mirrors the [MetaCubeX/meta-rules-dat](https://github.com/MetaCubeX/meta-rules-dat) GitHub repository, designed for deployment on Tencent Cloud EdgeOne Pages.

## How It Works

1. **GitHub Action**: Runs daily (or manually) to sync files
   - Downloads latest release assets from MetaCubeX/meta-rules-dat
   - Generates a JSON manifest of available files
   - Commits the files to the `main` branch

2. **Frontend**: Displays available files with download links
   - Fetches `/files.json` to get the file list
   - Shows file names, sizes, and last update time
   - Provides direct download links

3. **Deployment**: Tencent EdgeOne Pages automatically builds and deploys
   - Monitors the `main` branch for changes
   - Runs `pnpm build` to generate static files
   - Deploys the built site

## Local Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Deployment to Tencent Cloud EdgeOne Pages

1. Fork this repository
2. Enable GitHub Actions in your fork
3. Connect your repository to Tencent Cloud EdgeOne Pages
4. Configure build settings:
   - Build command: `pnpm build`
   - Output directory: `dist`
5. The GitHub Action will automatically sync files daily

## Manual Sync

Go to the "Actions" tab in your repository and run the "Sync Rule Files" workflow manually.
